# 文档审核助手

## 新增功能

### ③ 版本迭代记录
- 学生修改文档后再次上传时，AI会自动对比新旧版本
- 重点评价**修改是否有效、进步了多少**
- 显示版本差异（进步↑ / 退步↓ / 持平→）
- 评分变化用色彩标识（绿色增加、红色减少）

### ④ 知识库
- **教学文档**：上传后自动保存，支持多文件上传，提交时合并理解
- **作业文档**：每次审核自动保存，支持版本对比
- 支持搜索、加载、删除文档
- 可导出知识库为文本文件
- 最多保存50个文档、每个文档最多10个版本

### ⑤ 深度学术分析
在审核完成后，可使用以下三个深度分析功能：

#### ❓ 预设质疑生成
- 针对论文核心论点，AI模拟学术辩论
- 生成同行专家可能提出的尖锐质疑
- 提供回应方向参考

#### ⚠️ 薄弱环节压力测试
- 自动检测"断言多、证据少"的句子
- 用红色标注并给出改进建议

#### 📊 学术史立场可视化
- 分析论文在学术光谱中的位置
- 支持多维度分析（如：信古-疑古、经学今古文之争等）
- 标注立场并警示潜在论证矛盾

### ⑥ 经典互证参考
当论文涉及经典文献时，自动提供以下参考：

#### 📚 内证互参
- 检测论文中引用的经典（如《论语》《孟子》等）
- 自动列出该书中论同一主题的其他章节
- 例：讨论《论语》"仁"时，提示109处相关章节

#### 📖 训诂释义
- 检测关键词（如"信""义""德"等）
- 提供《说文解字》释义、段玉裁注
- 标注本义与引申义的区别
- 警示常见误解

#### 📚 跨文献关联
- 检测论文讨论的主题（如"性善论""大同"等）
- 推荐其他经典的相关论述
- 帮助学生发现文献间的关联与差异

## 安全性与隐私

- **所有数据仅保存在浏览器本地**（localStorage）
- 不上传到任何服务器
- 存储空间限制2MB，超出自动清理旧数据

## 使用方法

1. 双击打开 `index.html`
2. 输入智谱AI API Key（申请地址: https://open.bigmodel.cn/）
3. 在"提交教学文档"标签上传审核标准（支持多选文件）
4. 点击"理解文档"，AI会整合所有教学文档
5. 在"审核文档"标签上传学生作业
6. 相同文档名称会被识别为新版本，AI会自动对比
7. 审核完成后可使用"深度学术分析"功能
8. 在"经典互证参考"区域可一键检测文本中的经典引用

## 支持格式

- ✅ .txt .md .docx 直接读取
- ⚠️ .pdf 复制粘贴内容

## 内置数据

- 论语、孟子、礼记、周易、尚书、诗经的内证索引
- 25个核心词的训诂数据（说文解字+段注）
- 18个跨文献主题关联推荐

## 界面说明

| 标签 | 功能 |
|------|------|
| 📖 提交教学文档 | 上传审核标准（可多选），提交后自动保存到知识库 |
| 🔍 审核文档 | 上传学生作业，支持版本对比和深度分析 |
| 📚 知识库 | 管理教学标准和作业文档 |
